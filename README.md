# FMCG Profitability & Risk Analytics Dashboard
This project analyzes FMCG category-wise profitability and risk exposure using SQL and Power BI.

## Business Objective
- Identify high-risk categories with low profit margins  
- Track revenue contribution across regions  
- Provide action-based recommendations for management  

## Tools Used
- SQL (Data Preparation)
- Power BI (Dashboard & KPIs)
- DAX (Profitability Measures)
